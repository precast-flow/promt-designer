# Entegrasyon: üretim ve beton santrali

## Veri köprüleri

1. **Üretim emri / döküm** → numune kaydında seçilebilir kaynak (veya otomatik öneri).
2. **Seçilen reçete** → numune ve test sonuçlarında varsayılan beton tanımı.
3. **Santral operatör ekranı** (`12/06`) → “Bu döküme ait numune oluştur” kısayolu (mock).
4. **Slump hızlı giriş** → döküm id ile santral çıkışında ilişkilendirme.

## Çakışma ve uyarılar

- Döküm tamamlandıktan sonra geç girilen slump — izin ver / uyarı (ürün kararı).
- Üretim “kalite bekliyor” durumu — 12’deki üretim–kalite bitişik modeli ile uyumlu.
