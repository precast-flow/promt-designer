# Test kataloğu — beton / agrega ve santral bağlamı

Amaç: Hem **laboratuvar** hem **beton santrali sahasında** anlamlı olan test ve kontrolleri tek sözlükte toplamak. Gerçek standart kodları firma ülkesine göre özelleştirilebilir.

## Kategori A — Taze beton (saha / santral)

| Test | Türkçe yaygın ad | Tipik ölçüm | Not |
|------|------------------|-------------|-----|
| Slump | **Çekme (slump)** | mm veya cm | Santral çıkışında sık |
| Hava içeriği | Hava % | % | |
| Yoğunluk / birim ağırlık | kN/m³ veya kg/m³ | | |
| Sıcaklık | °C | | |

## Kategori B — Sertleşmiş beton (laboratuvar)

| Test | Açıklama | Not |
|------|----------|-----|
| Basınç dayanımı | Küp/silindir, yaş günü (7/28) | MPa |
| Esneklik / diğer mekanik | Ürün politikasına göre | |

## Kategori C — Agrega / diğer (opsiyonel)

| Test | Not |
|------|-----|
| Nem, elek analizi | İleride genişletilebilir |

## “Santralde kullanılabilir” rapor / form

- **Slump fişi** / **Sevkiyat öncesi kontrol** — taze beton.
- **Günlük santral özet** — döküm bazlı bağlantı (üretim `06` ile).
- **Basınç sonuç özeti** — yaş gününe göre tablo.

Her test tipi için **zorunlu alan seti** UI prompt’larında genişletilir.
