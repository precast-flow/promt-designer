# Açık kararlar (kalite)

1. Numune kimliği barkod ile mi, sadece sistem id ile mi?
2. Basınç için 7 ve 28 gün zorunlu mu, ek yaşlar var mı?
3. Onay için tek kişi mi, çift onay mı?
4. Müşteriye rapor teslimi portal mı, e-posta mı?
5. Periyodik raporlar otomatik üretilsin mi, manuel tetik mi?
6. Ulusal standart etiketleri (TS EN vb.) şablonda sabit mi, firma bazlı mı?
